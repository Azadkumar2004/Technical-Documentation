Variables

Variables

  • Store values: Represent numbers, text, or other data.
  • Declared using names: Follow naming conventions (lower case with underscores).

name = "Alice" age = 30 is_human = True # Boolean (True or False) pi = 3.14159

Variable Scope

  • Local variables: Exist within functions, accessible only within that function.
  • Global variables: Defined outside functions, accessible everywhere.

Python

def greet(): message = "Hi!" # Local variable print(message) greet() # Output: "Hi!" # print(message) # Error: NameError: name 'message' is not defined

Global Variables

Python

global my_variable # Declare as global to modify from outside def set_value(): global my_variable my_variable = 10 set_value() print(my_variable) # Output: 10

Control Flow

Controlling the flow of your program

Control flow statements determine the order in which your program's instructions are executed. Some common types of control flow statements include:

  • **if statements:** Execute code based on a condition being true.
  • **else statements:** Execute code if the condition in an if statement is false.
  • **for loops:** Repeat a block of code a specific number of times.
  • **while loops:** Repeat a block of code as long as a condition is true.
Operators

Performing Calculations and Comparisons

Operators are symbols used to perform various operations on values in your program. Here are some common types:

  • Arithmetic operators:** Perform calculations like addition,subtraction, multiplication, division, and modulo.
    • Example: x = 5 + 3 (result: x is 8)
    • Example:y = 10 // 3 (result: y is 3, floor division discards the remainder)
    • Example: `z = 10 % 4 (result: z is 2, modulo gives the remainder)
  • Comparison operators: Compare values and return True or False.
    • Example:a = 10 == 5 (result: False)
    • Example: b = 15 > 10 (result: True)
    • Example: c = "hello" != "world" (result: True)
Getting User Input

Python allows you to take input from the user during program execution using the `input()` function. This function prompts the user for input and returns it as a string.

Example:

name = input("Enter your name: ")

print("Hello,", name + "!")
Adding Comments to Your Code

Comments are lines of text ignored by Python when running the program. They are used to explain the code's purpose, logic, or specific parts. They are essential for:

  • Improving code readability and maintainability (for yourself and others).
  • Documenting the code's functionality.

Types of Comments:

  • Single-line comments: Start with `#` (hash symbol).
    Example:

    # This line calculates the area of a rectangle.
  • Multi-line comments: Use triple quotes (''' or """).
    Example:

    """ This code block calculates the area and perimeter of a rectangle. It takes the length and width as input from the user. """
